# **App Name**: FoodSnap

## Core Features:

- Image Upload: Allow users to upload food images via the UI.
- Dish Identification: Use AI to identify the food dish from the uploaded image.
- Recipe API Integration: Connect to a free recipe API (e.g., Spoonacular, TheMealDB) to fetch recipes.
- Recipe Display: Display recipe steps and ingredients in a readable format in the UI.
- Firestore Integration: Store image metadata and related recipe data in Firestore to maintain context.
- Loading Animation: Add a loading animation while the image is being processed and recipe details are being fetched.
- Recipe Suggestion: Implement a tool powered by generative AI to suggest recipes if the API fails or based on image specifics.

## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F) to evoke feelings of home cooking and comfort.
- Background color: Soft beige (#F8F0E3) to provide a neutral and welcoming backdrop.
- Accent color: Muted olive green (#81B29A) for complementary contrast and a touch of freshness.
- Font pairing: 'Belleza' (sans-serif) for headlines and 'Alegreya' (serif) for body text, providing a balance of style and readability.
- Use food-related icons (knife, fork, plate, etc.) to enhance user navigation and experience.
- Implement a clean and organized layout, ensuring easy navigation and readability of recipe steps.
- Subtle animations when transitioning between image upload, dish identification, and recipe display.
