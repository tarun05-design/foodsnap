# FoodSnap – AI-Based Food Recognition & Recipe Generator

## 📌 Overview

FoodSnap is a full-stack AI web application that analyzes food images and automatically generates recipes, ingredients, instructions, and nutritional information.

It combines computer vision APIs with generative AI to deliver intelligent food insights.

---

## 🎯 Key Results

- Automated recipe & nutrition generation
- Reduced manual data handling by **~40%**
- Integrated AI APIs for real-time responses
- Built authentication & database system using Firebase

---

## 🧠 System Architecture

1. User uploads food image
2. Image processed via AI recognition
3. Google Gemini generates:
   - Recipe
   - Ingredients
   - Cooking instructions
   - Nutritional values
4. Firebase stores user data & history

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Firebase (Auth + Firestore)
- Google Gemini API
- REST APIs

## 🌐 Features
